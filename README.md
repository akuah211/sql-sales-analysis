# SQL Sales Analysis

## Overview

This project involved analysing retail sales data using SQL and SQLite. The objective was to explore sales and profit performance across different product categories and regions by writing SQL queries and interpreting the results.

## Skills Demonstrated

* SQL
* SQLite
* Data Analysis
* Business Analysis
* Data Aggregation
* Query Writing
* GROUP BY
* SUM()
* ORDER BY

## Tools Used

* SQLite
* DB Browser for SQLite
* GitHub

## Conclusion

Through this analysis, I identified key trends in sales and profitability across the dataset. The results showed that Technology was the strongest-performing category in terms of both revenue and profit, while the West region generated the highest overall profit. This project strengthened my ability to use SQL to extract, analyse and interpret business data to support decision-making.
